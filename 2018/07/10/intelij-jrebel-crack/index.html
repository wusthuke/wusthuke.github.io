Jrebel热部署插件破解

简介

jrebel 升级到 2018.1 之后，Twitter 和 Facebook
账号不能再继续免费使用，见官网通知

可以通过自己搭建 License Server 来进行破解，参考: jrebel license server 激活方法

License Server搭建

  1. license Server代码准备

    • 方案一： GitHub下载源码，手工编译。
    • 方案二(推荐)： 百度网盘下载, 密码: 4y18

  2. Run Server

到这里，License Server 的搭建工作就完成了。准备开始 Intelij 的激活.

Intelij 安装jrebel插件并激活

  1. 下载 jrebel, 下载地址: intelij-jrebel-plugin下载
  2. intelij 从本地安装
  3. 激活jrebel, 其中，激活方式选择： license service

    server 地址： http://localhost:1008/e6271e94-6a6e-4ae5-9123-0a4e84c1e52a

    mail: 任意一个有效格式邮箱都可以.

GUID在线生成器 ： Server地址的 tokenName 生成器.

到这里就激活成功， 可以正常使用了。
