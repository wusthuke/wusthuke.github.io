SpringBoot启动及定制化开发

本文来分析SpringBoot启动过程，以及部分可定制化开发的点。主要可以分为下面几种：

  1. 初始化器 (Initializer)
  2. 监听器（Listener）
  3. 容器刷新后置Runners(ApplicationRunner或者CommandLineRunner接口的实现类)
  4. 启动期间在Console打印Banner的具体实现类
